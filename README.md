# PokeRogue Cheat Menu
A cheat menu for [PokeRogue](https://pokerogue.net) — install via Tampermonkey and press **`F2`** anywhere on the site to open it.

## Features
### 🥚 Eggs
* **Unlimited Vouchers** - set all voucher types to any amount (bulk or invididually)
* **Unlimited Egg Storage** - removes the 99-egg storage cap so Gacha pulls are never blocked
