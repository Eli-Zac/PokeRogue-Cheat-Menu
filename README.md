# PokeRogue Cheat Menu
A cheat menu for [PokeRogue](https://pokerogue.net) — install via Tampermonkey and press **`F2`** anywhere on the site to open it.

## Features

### 🥚 Eggs
* **Voucher Counts** — set Regular, Plus, Premium, and Golden vouchers individually, bulk-set to 999, or zero them all out
* **Remove Egg Limit** — toggle off the 99-egg storage cap so Gacha pulls are never blocked
* **Give Eggs by Type** — add up to 500 eggs of any type at once
* **Delete All Eggs** — clear your entire egg storage in one click

### ⚡ Pokémon
* **Catch All Pokémon** — instantly mark every Pokémon as caught in your Pokédex
* **Restore Default Pokémon** — revert your Pokédex back to its state when the page first connected (with confirmation)

### 💰 Gold
* **Set Currency** — set your gold to any exact amount
* **Add Currency** — add any amount on top of your current balance
* **+10,000 Shortcut** — add 10,000 gold in one click

### 💾 Save Data
* **Export Save** — download your full game progress (Pokédex, starters, eggs, vouchers, achievements, stats, and more) as a JSON file
* **Import Save** — load a previously exported JSON file to restore all persistent progress, then auto-save to the server

### 📊 Game Stats
* **Live Statistics** — read-only live view of all game counters including battles, catches, eggs hatched, play time, and more — updates every second
